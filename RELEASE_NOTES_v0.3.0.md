# Aitri v0.3.0

## Focus
Stabilized Spec-Driven Core (Guardian Phase)

## Added
- `aitri status`
- `aitri validate` with basic traceability enforcement
- Spec → Backlog → Tests discipline
- Architecture documentation
- V1 Scope Lock defined

## Enforced Principles
- No downstream artifacts without approved spec
- Traceability required before validation passes
- Deterministic CLI behavior
- Human approval required before implementation

## Not Included (Deferred)
- AI orchestration
- Multi-agent automation
- Semantic validation
- External integrations (Jira, GitHub Apps, etc.)
- UI / Dashboard

## Positioning
Aitri is now a stable spec-driven SDLC guardian.
Generation and intelligent assistance will evolve in future minor versions.
